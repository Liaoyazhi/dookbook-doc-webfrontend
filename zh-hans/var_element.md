# `<var>`

`<var>` 标签表示变量的名称，或者由用户提供的值。

|  |  |
| :-- | :-- |
| **允许内容** | 短语内容 |
| **标签闭合** | 开始标签和结束标签都是必需的 |
| **允许的父级元素** | 任何接受短语内容的元素. |
| **规范元素** | HTML，"`var`元素" |

## 属性

本元素仅支持[全局属性](/zh-hans/webfrontend/HTML_Global_Attributes).

默认样式

```css
var {
  font-style: italic;
}
```

## 示例

```html
<p> A simple equation: <var>x</var> = <var>y</var> + 2 </p>
```
